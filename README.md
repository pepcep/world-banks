# World banks

Go Library for retrieving world banks.

## Quick start

To install:

```
go get github.com/pepcep/world-banks
```

#### Usage

There are several ways of utilizing this package. One of the ways include:

```go
func main() {
	banks, err := wb.GetNigerianBanks()
}
```
